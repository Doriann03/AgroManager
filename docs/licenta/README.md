# Lucrare de licență - AgroManager

Titlu: **Aplicație web pentru gestionarea activităților unei ferme agricole**

Acest folder conține textele de lucru pentru partea scrisă a lucrării de licență. Conținutul este gândit pentru a fi redactat inițial în Markdown, apoi transferat în documentul Word final creat pe baza template-ului facultății.

## Convenție de lucru

- Textele sunt redactate academic, la persoana a III-a.
- Structura profesorului are prioritate: analiza problemei, cerințe, proiectare, implementare, securitate, testare, descriere/comparații, bibliografie.
- Funcțiile aflate încă în lucru pot fi tratate ca parte din versiunea finală a aplicației, dar sunt urmărite separat în `00_dezvoltari_de_aliniat.md`, pentru ca aplicația practică să ajungă la același nivel înainte de predare.
- Capturile de ecran vor fi realizate manual și inserate în documentul Word final, conform listei din `00_capturi_ecran.md`.
- Codul citat în lucrare trebuie ales din implementarea reală, după stabilizarea modulelor.

- `Dorian.docx` - documentul Word final de lucru, completat manual pe baza drafturilor Markdown.

## Observație

Pentru documentul Word final se vor respecta regulile template-ului: format A4, Cambria 12, spațiere 1,15, margini de 2,54 cm, titluri cu stiluri Heading, figuri/tabele numerotate și bibliografie cu trimiteri de tip [1].
